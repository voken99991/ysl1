# YSL Website

Open `index.html` to preview.

Included:
- Single-screen desktop homepage
- Separate pages for Fixtures, Standings, Playersheet, News, About, Admin and Dashboard
- Light mode and dark purple mode
- Theme preference saved in the browser
- Three editable upcoming fixture cards

Local preview:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
